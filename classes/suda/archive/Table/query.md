## 例子


请尽量避免使用此函数
其中 #{user} 表示user表，加上 #{} 框架会自动处理浅醉

```php
$table->query('select * from #{user} where id > :id',['id'=>2]);
```

可以使用 @table@ 代指本表

```php
$table->query('select * from #{@table@} where id > :id',['id'=>2]);
```
