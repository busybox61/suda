## 例子


条件可以为键值对也可以为特殊条件

**键值对**

更新 ID 为3 的name 为 $name 的值

```php
$table->update(['name'=>$name],['id'=>3]);
```

**条件**

更新 ID>3 的name 为 $name 的值

```php
$table->update(['name'=>$name],'id > :id ',['id'=>3]);
```
