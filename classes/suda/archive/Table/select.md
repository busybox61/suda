## 例子


相当于 select 语句，返回一个 SQLQuery类

查询：取一列name，当id为2的时候

```php
$table->select(['name'],['id'=>2])->fetch();
```

查询：取所有列name，当 status = 2的时候

```php
$table->select(['name'],['status'=>2])->fetchAll();
```

查询：取多列，当 id >2 的时候

```php
$table->select(['name'],'id > :id',['id'=>2])->fetchAll();
```
